## Angular TS base client

### Quick start
```
npm install
ng serve --port=8000
```

### Get started
* Install Angulat CLI, initiate project and heleper dev tool.
```
npm install -g @angular/cli
```

* Init project 
```
ng new web-client // --skip-install to install all dependencies later
```
